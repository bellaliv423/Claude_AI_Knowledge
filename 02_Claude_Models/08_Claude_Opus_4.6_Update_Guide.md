## Part 2: 신규 기능 하나씩 파헤치기

### 기능 1: Adaptive Thinking (적응형 사고)

**이전**: Extended Thinking을 **켜거나 끄는** 이진 선택만 가능
**이제**: Claude가 **알아서 판단!** 복잡한 질문 = 깊이 생각, 간단한 질문 = 빠르게 답변

쉽게 비유하면:
- **이전** = "풀파워 모드 ON/OFF" 스위치밖에 없음
- **이제** = Claude가 상황 보고 **적절한 세기로 자동 조절**

> **비유**: 에어컨 온도를 수동(ON/OFF)으로 조절하다가, 이제 **자동 모드**가 생긴 거예요!

---

### 기능 2: Effort Controls (노력 수준 제어)

4단계 레벨로 **지능 vs 속도 vs 비용**의 균형을 조절할 수 있어요.

| 레벨 | 특징 | 추천 상황 | 비용/속도 |
|:--:|------|---------|---------|
| **Low** | 최소 추론, 빠른 응답 | 간단한 질문, 분류 | 가장 저렴, 가장 빠름 |
| **Medium** | 적절한 추론 | 이메일 작성, 번역 | 균형 |
| **High (기본)** | 필요시 확장 사고 | 보고서 분석, 코딩 | 높은 품질 |
| **Max** | 최대 추론 능력 | 연구, 복잡한 추론 | 가장 비쌈 |

> **실전 팁: 언제 어떤 레벨?**
> - 간단한 번역/요약 → **Low** (빠르고 저렴)
> - 바이어 이메일 작성 → **Medium** (적절한 품질)
> - 매출 분석 보고서 → **High** (깊은 분석 필요)
> - MBA 논문/학술 연구 → **Max** (최대 추론 능력)

## Part 7: 자주 묻는 질문 (FAQ)

**Q1: claude.ai에서 Pro 플랜이면 Opus 4.6 바로 쓸 수 있나요?**
> 네! claude.ai에서 모델 선택 시 Opus 4.6을 선택하면 바로 사용할 수 있어요.

**Q2: Claude in Excel/PPT는 Max 플랜 안 쓰면 못 쓰나요?**
> 네, Max ($100/월) 이상이 필요해요. 하지만! **claude.ai 채팅에서도 엑셀/PPT 파일을 생성**할 수 있어요. Pro ($20)로도 충분합니다.

**Q3: Adaptive Thinking은 내가 설정해야 하나요?**
> 아니요! **자동으로 작동**합니다. 기본 Effort 레벨(High)에서 Claude가 필요할 때만 깊이 생각해요. 별도 설정 불필요!

**Q4: Context Compaction은 어떻게 켜나요?**
> 현재 베타 기능으로 API에서 사용 가능합니다. claude.ai 채팅에서는 자동으로 적용되는 경우도 있어요.

**Q5: Agent Teams는 비개발자도 쓸 수 있나요?**
> Agent Teams는 Claude Code(터미널) 전용이라 개발자용이에요. 비개발자는 **claude.ai 채팅**에서 동일한 작업을 할 수 있어요.

**Q6: 1M 토큰 컨텍스트는 무료인가요?**
> 200K까지는 기본 가격, 200K 초과 시 프리미엄 가격($10/$37.50 per 1M)이 적용됩니다.

**Q7: Opus 4.6은 Opus 4.5보다 뭐가 좋아졌나요? (한 줄 요약)**
> **코딩 더 잘하고, 더 오래 기억하고, 더 정확하게 추론하고, 3배 더 저렴해졌어요!**

## Part 8: 핵심 정리

### 핵심 인사이트 3가지

**1. Claude는 이제 "도구"가 아니라 "팀원"**
Agent Teams로 여러 AI가 팀으로 협업하고, Excel/PPT 안에서 직접 작업하며, 자율적으로 계획→실행. 단순한 챗봇이 아니라, **진짜 팀원처럼 일하는 AI!**

**2. 비개발자의 시대가 왔다**
Claude in Excel/PPT는 코딩 없이 앱 안에서 바로 사용. claude.ai 채팅에서도 엑셀/PPT/워드 파일 직접 생성. **"코딩 못해서 AI 못 써요"는 이제 옛날 이야기!**

**3. 가격 혁명**
최고급 모델(Opus)이 이전 대비 **3배 저렴**. 더 적은 비용으로 더 강력한 AI 활용 가능!
