## AI 협업 규칙 / AI Collaboration Rules

### Claude와 협업 시 준수사항

1. **CLAUDE.md 활용**
   - 프로젝트마다 CLAUDE.md 파일 유지
   - 실수 발생 시 즉시 기록
   - 팀 공유 메모리로 활용

2. **Plan Mode 우선**
   - 복잡한 작업 전 반드시 계획 수립
   - 명시적 의도와 제약 조건 설정
   - 원치 않는 연쇄 변경 방지

3. **피드백 루프 제공**
   - 테스트/빌드로 검증 루프 구축
   - Claude에게 자신의 작업 검증 방법 제공
   - 품질 2-3배 향상 효과

4. **버전 관리**
   - 모든 변경사항 Git 커밋
   - 의미 있는 커밋 메시지 작성
   - 정기적인 GitHub 푸시

### 문서 작성 규칙

1. **이중 언어**: 한국어 + 繁體中文
2. **형식 일관성**: 기존 문서 스타일 유지
3. **예제 포함**: 실행 가능한 코드 예제 필수
4. **버전 기록**: 생성일, 업데이트일 명시
